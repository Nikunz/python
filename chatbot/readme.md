Pre-requisites 
 - tensorflow
 - keras
 - pickle
 - nltk

   ```
   pip install tensorflow, keras, pickle, nltk
   ```
 Other dependencies required were "punkt" and "wordnet"
   ```
   python3
   >>> import nltk
   >>> nltk.download('punkt')

   python3
   >>> import nltk
   >>> nltk.download('wordnet')
```
